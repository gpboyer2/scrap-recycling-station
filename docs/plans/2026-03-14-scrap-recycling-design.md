# 废品回收站项目设计

## 项目概述

为废品回收站创建一个简单的前后端项目，一期功能聚焦于信息展示。

## 技术栈

- 前端：Vue 3
- 后端：Node.js + Express
- 数据库：SQLite
- 价格维护：JSON 配置文件

## 项目结构

```
废品回收/
├── server/                 # 后端
│   ├── index.js           # 入口
│   ├── config.json        # 价格配置（修改后重启生效）
│   └── data.db            # SQLite 数据库
├── client/                 # 前端 Vue
│   └── src/
│       ├── App.vue
│       └── main.js
└── package.json
```

## 核心功能

### 首页展示
- 回收站名称
- 地址（带导航链接，点击跳转地图APP）
- 联系电话

### 价格表
- 废品名称
- 单位
- 今日回收价

### UI 风格
- 极简风格：白底黑字，清晰的价格表格
- 移动端优先：大字体、大按钮、清晰的交互区域

## 废品类型预设

| 类型 | 单位 | 示例价格（元） |
|------|------|----------------|
| 黄纸板 | 斤 | 0.8 |
| 书本纸 | 斤 | 0.6 |
| 报纸 | 斤 | 0.7 |
| 塑料瓶 | 斤 | 1.2 |
| 易拉罐 | 斤 | 3.5 |
| 废铁 | 斤 | 1.0 |
| 废铜 | 斤 | 25.0 |
| 废铝 | 斤 | 5.0 |

## 价格维护方式

修改 `server/config.json` 文件中的价格数据，重启服务即可生效。

## API 设计

- GET /api/info - 获取回收站基本信息
- GET /api/prices - 获取今日废品价格
